Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Announcements

Retrieve a list of announcements.

공지사항 목록을 가져옵니다. 각 게시물은 ID, 제목, 메인 표출 여부, 생성 날짜를 포함합니다.

Example request:
curl --request GET \
    --get "http://localhost/api/announcement?search=&page=" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/announcement"
);

const params = {
    "search": "",
    "page": "",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Endpoints

POST api/inquiry

Example request:
curl --request POST \
    "http://localhost/api/inquiry" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"minima\",
    \"contact\": \"eum\",
    \"inquiry_category\": \"labore\",
    \"email\": \"cyril.sipes@example.com\",
    \"message\": \"repellat\",
    \"agreement\": false
}"

const url = new URL(
    "http://localhost/api/inquiry"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "minima",
    "contact": "eum",
    "inquiry_category": "labore",
    "email": "cyril.sipes@example.com",
    "message": "repellat",
    "agreement": false
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
